# Team Task Manager Frontend

React + Vite frontend for Team Task Manager.

## Setup

1. Install dependencies:
   - `npm install`
2. Create `.env`:
   - `VITE_API_URL=http://localhost:5000/api`
3. Start app:
   - `npm run dev`

## Pages

- Login / Signup
- Dashboard
- Projects
- Project Details (team + tasks)
- Tasks
